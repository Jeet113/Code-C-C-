# Code-C&C++

Solutions to programming problems and practice exercises in C and C++. Most files are named after the problem ID (for example, `1238A.cpp`, `110A.c`) and contain a single standalone solution.

## Contents
- A large collection of single-file solutions in C and C++
- Some utility or practice files (sorting, searching, small demos)

## File Naming
- Numeric prefixes typically map to problem IDs
- Suffixes like `A`, `B`, `C` indicate problem variants
- Mixed `.c` and `.cpp` files depending on language used

## Build and Run
Each file is independent. Compile the file you want to run.

### C (GCC)
```bash
gcc -O2 -std=c11 -Wall -Wextra -pedantic file.c -o app
./app
```

### C++ (G++)
```bash
g++ -O2 -std=c++17 -Wall -Wextra -pedantic file.cpp -o app
./app
```

## Notes
- Input and output follow the corresponding problem statements.
- These solutions are intended for learning and competitive programming practice.

## Sources
Most problems are from online judges (for example, Codeforces). File names usually match the problem ID.
